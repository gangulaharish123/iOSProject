# Overview
This application demonstrates to store the password in secure place i.e keychain and also show in view. 
- Xcode 14.0.1
- Swift 5

The application accepts the password from the user based on given criteria.
The password must be at least 8 characters long and include at least one uppercase letter, one lowercase letter, one number, and one special character.

- The unit test coverage is completed.
